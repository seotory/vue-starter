# 프로젝트 시작하기

nodejs, typescript, webpack, vue, karma, browser sync, phantomjs 합본입니다.

## 설치

`npm install`로 설치하지 마십시오. 아래의 명령어로 실행시켜 주세요. 이 명령어에는 2가지의 일을 포함하고 있습니다.

- npm 모듈 설치
- typescript의 d.ts 설치

```
npm run init
```

## 빌드

```
npm run build 
```

## 브라우저 테스트

```
npm run test
```
